## Install

- Node.js 18+ is recommended.

### NPM

To install via NPM:

```
npm install @isomedias/hybridforms
```

### Development

To install locally (for development):

```
git clone https://github.com/IcoDeveloper/hybridforms-js.git
cd hybridforms-js
npm install
```
